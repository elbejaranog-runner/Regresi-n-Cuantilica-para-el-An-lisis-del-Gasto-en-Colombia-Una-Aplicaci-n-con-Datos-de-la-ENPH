# Regresi-n-Cuantilica-para-el-An-lisis-del-Gasto-en-Colombia-Una-Aplicaci-n-con-Datos-de-la-ENPH
Esta regresión cuantílica se usa para estimar la curva de Engel que describe la relación entre el gasto total del hogar y el gasto en mercado en Colombia

Comentario: Para cargar usar el código que se encuentra en este repositorio se debe cargar las dos bases de datos que hacen parte de GEPH:

Gastos diarios urbanos - mercados
Viviendas y hogares

Ambos disponibles es la página del DANE https://microdatos.dane.gov.co/index.php/catalog/566/get_microdata
